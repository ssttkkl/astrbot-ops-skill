# AstrBot 插件开发参考文档

本目录包含 AstrBot 官方插件开发文档的完整副本。

## 文件说明

- `plugin.md` - 旧版插件开发指南（v4.5.7 之前）
  - 包含完整的 API 参考
  - 详细的代码示例
  - 所有功能的使用说明

- `plugin-new.md` - 新版插件开发指南（v4.5.7+）
  - 推荐阅读的最新指南
  - 与旧版内容基本一致
  - 持续维护更新

## 使用建议

1. **开发前必读**：在开发任何插件功能前，先查阅相关文档
2. **优先查看新版**：从 `plugin-new.md` 开始
3. **API 参考**：需要详细 API 说明时查看 `plugin.md`
